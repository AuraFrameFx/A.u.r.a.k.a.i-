## ğŸš¦ Getting Started

### Prerequisites

- Android 14+ device
- LSPosed Framework installed
- Root access (for full features)
- JDK 25 installed (builds with Java 24 bytecode)

### Installation

```bash
# Clone the repository
git clone https://github.com/AuraFrameFx/A.u.r.a.K.a.i_Reactive-Intelligence-.git

# Navigate to project
cd A.u.r.a.K.a.i_Reactive-Intelligence-

# Build the module (debug)
./gradlew :app:assembleDebug

# Or for release (with ProGuard optimization)
./gradlew :app:assembleRelease

# Install via ADB
adb install app/build/outputs/apk/debug/app-debug.apk
```

### Quick Setup

1. Enable the module in LSPosed Manager
2. Select target apps (or system-wide)
3. Reboot your device
4. Launch AuraKai
5. Meet Aura & Kai!
