### For Developers

**Accessing Gate Cards:**
```kotlin
// In Jetpack Compose

    painter = painterResource(R.drawable.gate_romtools),
    contentDescription = "ROM Tools Module Gate"
```

**In Documentation:**
```markdown

```

### For Designers

**Creating New Gate Cards:**

1. **Size:** 1920x1080px minimum (16:9 ratio)
2. **Format:** PNG with transparency
3. **Color:** Use module's assigned color from palette
4. **Style:** Abstract geometric patterns reflecting function
5. **Export:** Save as `gate_modulename.png` in appropriate `drawable/` folder
